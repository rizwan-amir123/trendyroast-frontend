# 🔥 TrendyRoast - AI Roast Generator

A fun web app that fetches **live trending news** and generates **savage, funny tweets** using Groq AI (Llama 3).

### Features
- Live trending news from NewsAPI
- AI-generated hilarious roast tweets
- Modern dark UI + fully responsive
- Direct integration with Groq AI (super fast)

### Tech Stack
- **Frontend**: Angular 17+ (Standalone Components)
- **Styling**: Bootstrap + Custom SCSS
- **AI**: Groq (Llama-3.3-70B)
- **News**: NewsAPI.org

```bash

# Create src/environments/environment.ts with your keys
